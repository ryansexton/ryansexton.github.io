Python Asteroids Game

Skills Used:
Python
Go to Code on GitHub
Description:
This is a Python coding project that I created for CISC108: Introduction to Computer Science 1 class. I used Thonny and the designer library to write the python script for the game. The objective of the game is to score as many points as possible, displayed in the left corner. You control a spaceship that moves in all directions using the arrow keys and can shoot lasers using the space bar. Collect stars (1 point) and rescue astronauts (5 points) while avoiding asteroids from above and alien interceptors from the sides of the screen. The coding is simple using when statements to run the game. All the items and characters are images with a hitbox. Objects are loaded into an array which has a cap to avoid crashing the game. You'll notice this if you don't destroy any asteroids.
