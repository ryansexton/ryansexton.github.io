Personal Finance Tracker

Skills Used:
Microsoft Excel
Power BI
Download Excel Spreadsheet
Download Power BI Dashboard
Description:
In this personal project I used Microsoft Excel and Power BI to track my personal finances. For privacy the values used in the Excel sheet are just sample values. The dashboard uses an excel shreadsheet as it's data source. The excel sheet categorizes values as either income, expenses, or savings, along with sub-categories the further specify the monthly transactions. The dashboard displays many useful visualizations that help me track my income, spending and saving habits. Among these visualizations is my total net worth, monthly transaction trends, and a breakdown of spending and savings. I can use a slicer to specify the date range of finances I want to view whether that be a few months or all time. Currently I have to manually imput my finances into the excel sheet but I'm looking into programming the excel sheet to read in a CSV of my bank statements at the end of each month. Download the Excel file and Power BI dashboard for yourself to start tracking your finances!
